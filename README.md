## 👩‍💻 Sobre mí

Soy desarrolladora web en formación, cursando **2º de DAW**, con más de 12 años de experiencia profesional previa que me han dado una perspectiva diferente: sé trabajar bajo presión, comunicarme con cualquier perfil y adaptarme rápido a entornos nuevos.

Lo que más me apasiona del desarrollo es el mundo de las **bases de datos** y todo lo que se puede automatizar con código. Aprendo rápido y siempre busco entender el porqué de las cosas.

- 🎓 CFGS Desarrollo de Aplicaciones Web — 2º año
- 💡 Intereses: bases de datos, automatización, backend
- 🌱 Aprendiendo: JavaScript · Laravel avanzado · REST · SOAP · AJAX · jQuery · Python
- 📍 Córdoba, España
- 🌍 Español nativo · Inglés básico
